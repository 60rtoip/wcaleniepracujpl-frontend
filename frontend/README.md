# Frontend (minimal)

This is a tiny demo React + Vite frontend for the Job Board backend.

Quick start:

```bash
cd frontend
npm install
npm run dev
```

Environment:
- `VITE_API_ORIGIN` — API origin, defaults to `http://localhost:8000`.

Implemented pages (very small): Home, Jobs list, Companies list, Login.
